# Aula(Recriando a Página do Instagram)

- Nesta aula utilizando comandos em HTML5 e CSS3 recriamos a página de login do instagram.
- Adicionar a página HTML5 e os respectivos arquivos.
- Identificar a quantidade de colunas necessárias(celular e pagina de login)
- Identificar a quantidade de containers(neste cenário foram 3)
- Com o CSS3 criar a estilização com a finalidade de clonar a página do instagram.
